RaLP (Register and Login Plugin) is a must-have security plugin for Minecraft servers, offering a seamless yet powerful authentication system to prevent unauthorized access.

Key Features:
- Secure Registration & Login – Players must register with a password and log in to play
- Password Hashing – Uses BCrypt for maximum security (no plaintext passwords!)
- Anti-Bruteforce Protection – Blocks players after too many failed attempts (Except mod)
- Auto-Login – Optional auto-login for trusted devices (Except mod)
- Multi-Language Support – Built-in translations (English, Russian, German, Spanish, French, Portuguese, Japanese, Chinese, Korean, Turkish, Italian, Polish)
- Session Management – Auto-logout after inactivity (Except mod)
